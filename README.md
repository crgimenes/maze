# Maze

Golang maze generator in text mode, inspired by the old BASIC code

```basic
10 PRINT CHR$(205.5+RND(1)); : GOTO 10
```

```lua
::_10:: io.write(math.random() < 0.5 and "/" or "\\"); goto _10
```
